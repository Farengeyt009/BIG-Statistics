# QC.vw_Stamping_Output

## Назначение

Представление возвращает список деталей, которые являются **финальным выходом цеха штамповки** — то есть деталь уже прошла все операции штамповки и готова к передаче на следующий участок (покраска, шелкография и т.д.).

---

## Контекст: зачем это нужно

Физически одна деталь может проходить штамповку в несколько операций, каждый раз получая новый артикул:

```
7.66.01...1  →  8.75.10...1  →  8.75.10...2  →  8.75.11...1
 (заготовка)    (штамп 1)       (штамп 2)         (покраска)
```

Нам нужен именно **8.75.10...2** — последний артикул перед переходом на следующий участок. Это важно для расчёта себестоимости: именно эта деталь входит в спецификацию следующего передела.

---

## Логика

**Шаг 1.** Из BOM (`vw_Import_BOM_Current`) берём все `GPNomencl_ID` (выпускаемые изделия) у которых группа работ (`Labor_TypeGroupeID`) = **Штамповка ТТ**.

Получаем множество всех деталей, которые когда-либо производились в рамках штамповки ТТ.

**Шаг 2.** Снова обращаемся к BOM и ищем строки, где эти изделия выступают как `Nomencl_ID` — то есть являются **входящим материалом** для следующей операции. Это означает, что деталь уже вышла из штамповки и используется где-то дальше.

**Шаг 3.** Убираем строки, где сам входящий материал (`a.Labor_TypeGroupeID`) тоже относится к штамповке ТТ. Такие строки означают промежуточные операции внутри штамповки — нас они не интересуют.

**Результат:** детали, которые вышли из штамповки и передаются на другой участок.
